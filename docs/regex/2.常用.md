# 常用

## 匹配括号内的值

`(?<=\()[^\)]+`

`(?<=\《)[^\》]+`

## 邮箱

`/^[a-z0-9]+([._\\-]*[a-z0-9])*@([a-z0-9]+[-a-z0-9]*[a-z0-9]+.){1,63}[a-z0-9]+$/`

## ip

`/^(\d{1,2}|1\d\d|2[0-4]\d|25[0-5])((\.(\d{1,2}|1\d\d|2[0-4]\d|25[0-5])){3}|(\.(\d{1,2}|1\d\d|2[0-4]\d|25[0-5])){5})$/`

## url

`/[a-zA-z]+:\/\/[^\s]/`

## 中文

`/^[\u0391-\uFFE5]+$/`
